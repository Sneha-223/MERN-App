## Running the code

* Run Mongo daemon:
```
sudo systemctl start mongod
sudo systemctl status mongod
```


* Run Express Backend:
```
cd backend/
npm install
npm start
```

* Run React Frontend:
```
cd frontend
npm install/
npm start
```

Navigate to [http://localhost:3000/](http://localhost:3000/) in your browser.
